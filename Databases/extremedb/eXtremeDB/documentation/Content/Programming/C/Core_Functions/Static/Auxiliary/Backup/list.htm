mco_backup_list

List the contents of a backup file.

For an overview see page Incremental Online Backup in C

Prototype

 
    void	mco_backup_list(	/*IN*/ mco_db_h con, 
                    /*IN*/ char const* file_name, 
                    /*IN*/ mco_size_t* backup_list_size,
                    /*IN*/ mco_backup_info_t** backup_list,
                    /*IN*/ char* err_buf, 
                    /*IN-OUT*/ unsigned int * err_buf_sz)
 

Arguments

con A database connection used to get the name of the database put in the backup file
file_name The name of the backup file to list
mco_size_t* backup_list_size A pointer to mco_size_t variable that will get the number of records in the list returned
mco_backup_info_t** backup_list A pointer to variable that will point to an array of backup records descriptors. The routine allocates memory for the array internally and the memory needs to be returned to the pool using the mco_sys_free() API
err_buf A buffer for error description
err_buf_sz On input this specifies the size of err_buf. On return it contains the number of characters written into the buffer. If err_buf_sz is null the routine returns MCO_E_ILLEGAL_PARAM

Description

This function creates a list of backup records in the specified backup file. The list includes the name, type, label and other properties of the backup records. (See the Incremental Backup and Restore page for further details.)

Return Codes

MCO_S_OK The list was successfully returned
MCO_E_DISK_OPEN Could not open the backup file

Example

     
    {
        mco_db_h con;
        MCO_RET rc;
        char * filename = “backup.bak”;
        char * label = “a label”;
 
        /* connect to a database */
        rc = mco_db_connect( db_name, &con );
 
        /* backup the database */
        rc = mco_backup_list( con, filename, ... );
    }
