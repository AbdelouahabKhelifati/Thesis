mco_async_event_release_all

Unblock all the threads that are waiting on all events for the specified database.

Prototype

 
    MCO_RET	mco_async_event_release_all(	/*IN*/ mco_db_h db );
 

Arguments

db The database handle that was established by mco_db_connect()

Description

This function unblocks all the threads that are waiting on all events for the specified database.

Return Codes

MCO_S_OK Events were released successfully
MCO_E_INVALID_HANDLE The database handle is invalid
MCO_S_EVENT_RELEASED Events were already released
