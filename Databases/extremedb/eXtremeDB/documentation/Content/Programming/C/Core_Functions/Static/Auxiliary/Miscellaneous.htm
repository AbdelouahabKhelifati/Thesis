Miscellaneous Functions

The following APIs perform specialized functions :

mco_db_register_collations Register collation functions with the runtime
mco_system_get_current_time

Retrieve the operating system time
mco_ticks2time Convert datetime between the application's and database's resolution; convert
mco_time2ticks Convert datetime between the application's and database's resolution; convert
mco_core_load Loads eXtremeDB libraries dynamically
mco_core_is_loaded Return true if the core eXtremeDB dynamic libraries are loaded
