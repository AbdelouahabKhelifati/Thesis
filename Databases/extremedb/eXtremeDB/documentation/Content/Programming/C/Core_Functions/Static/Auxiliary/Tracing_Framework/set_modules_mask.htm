mco_trace_set_modules_mask

Set the eXtremeDB modules to be logged.

For an overview see page Tracing Framework in C

Prototype

 
    uint8 mco_trace_set_modules_mask(uint8 new_mask);
 

Arguments

new_mask

The bitmask that determines which eXtremeDB modules are enabled for trace output. The possible flags are defined in enum mco_trace_module

Description

This function determines the eXtremeDB modules to be logged. Please note argument new_mask is a bit mask, and its values must be shifted when used. For example:

 
    mco_trace_set_modules_mask(1<<MCO_TRACE_TMGR | 1<<MCO_TRACE_TL);
     

In addition the following alias can be used to cause all modules to be traced:

 
    new_mask = MCO_TRACE_MASK_ALL;
     

Return Codes

uint8 The previous trace modules mask
