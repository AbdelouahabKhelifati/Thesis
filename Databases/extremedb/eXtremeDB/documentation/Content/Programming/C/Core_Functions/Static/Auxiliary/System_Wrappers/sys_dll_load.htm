mco_sys_dll_load

Load an external module (dll, so or etc.) library and return the entry point of the specified function.

Prototype

 
    void* mco_sys_dll_load(char const* library, char const* func);
 

Arguments

library The name of the library (dll, so or etc.) file to load dynamically
func The name of the function entry point to return

Description

This function loads an external module (dll,so or etc.) library, finds in it the function func and returns the entry point of the function.

(Note that this is a platform-independent interface for a dynamic loader if it is supported by the underlying OS).

Return Codes

The a pointer if the operation succeeded and NULL otherwise.
void * A pointer (the entry point of the specified function) if successful
NULL The load operation failed
