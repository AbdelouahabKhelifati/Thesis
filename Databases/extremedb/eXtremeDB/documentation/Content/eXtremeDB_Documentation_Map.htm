eXtremeDB Documentation

To view this eXtremeDB online documentation simply expand and click on the topics in the Table of Contents (ToC) pane to the left, then navigate to related topics using the links provided. (For example, click on this link to view New Features, then use the browser's back arrow to return to this page.)

The search box in the upper right corner of the browser page is an easy way to find topics of interest. To begin, type a word, a partial word or multiple words and press "Enter" or click on the magnifying glass icon to the right. Use quotes to search a multi-word phrase; for example "batch insert" will return only instances of the two words together whereas without the quotes all instances of either "batch" or "insert" will be found. (Note that there are some common words that cannot be searched, like "for", "because" and "the". Please see the complete list of unsearchable words here.)

Click on any topic title in the search results to open the page of interest. The page is displayed and its associated ToC entry is highlighted in the left pane. The right and left arrows at the top left corner of the pane navigate forward and backwards through the neighboring ToC entries, while the "ToC path" in the top of the page displays the current position in the Table of Contents. (Clicking on an element in this path is another way to navigate through the Table of Contents.)

Getting Started

For developers new to eXtremeDB we recommend viewing the Fundamental Concepts pages and the Getting Started pages for installation instructions and Quick Start tutorials to help you get up and running quickly.

User's Guides

Once familiar with eXtremeDB features, please view the eXtremeDB Embedded User's Guide or the eXtremeDB for HPC User's Guide for a range of topics giving detailed explanations and implementation examples.

Programming eXtremeDB

The Native APIs pages provide detailed Reference Guide information and many sample programs for each of the supported language interfaces: C, C++, Java, C# and Python.

The SQL APIs pages present xSQL and implementation details for embedded eXtremeSQL applications including examples using each of the supported languages.
