Analytics Functions Example 10: Group Aggregate

Group Aggregate Sum

Get the total volume for each month of the first quarter of 2013:

     
    SELECT symbol, 
        seq_search(day, 20130101, 20130331) as trade_date,
        volume@trade_date as "volume",
        seq_group_agg_sum(volume@trade_date, trade_date/100) as "month_vol"
    FROM Quote WHERE symbol='IBM';

We can analyze the output above as follows:

  • The output values for sequences trade_date and volume are divided by month for convenience.
  • The sequence month_vol shows the “group_aggregate sum“ for each of the months (calculated as trade_date/100, eg. 20130101/100 = 201301).
  • The volumes for the month of February are totaled as follows to produce the corresponding value 64146396 in sequence month_vol:
