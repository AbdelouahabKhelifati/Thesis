Analytics Functions Example 12: Sort

Sort Sequence Elements

Show close and volume values in descending order by volume:

     
    SELECT symbol, 
        seq_sort(volume, 'desc') as desc_volumes, 
        close@desc_volumes
    FROM Quote WHERE symbol='SYM0';
