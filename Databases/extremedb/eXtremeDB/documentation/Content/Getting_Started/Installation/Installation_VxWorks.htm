Installing eXtremeDB on VxWorks

After downloading, simply extract the contents of the archive file to a convenient directory. For example, suppose that you purchasedeXtremeDB High Availability for VxWorks version 6.9 with the Diab tool chain for applications on ARM target architecture, and you wish to use the root directory "eXtremeDB". Installing on a Windows host development environment, simply extract the contents of the zip file into directory eXtremeDB.

Now all of the software components necessary to build eXtremeDB C/C++ applications, as well as documentation and SDK samples, are available in the root directory eXtremeDB. After installation, your directory tree will look like the following:

 
    \eXtremeDB
        documentation 
        host 
        include 
        samples 
        target 
        license.txt
     

See Package Contents for an explanation of these contents.
